Pixel2HTML Interview Challenge
======
**pixel2html-final** is a interview challenge given by pixel2html.com. The site consists of 6 sections - header, hero, image/slider, pricing, companies & footer. 

## Technologies
* HTML5
* CSS3
* Bootstrap
* JavaScript
* jQuery
* Gulp
* NPM
* Slick.js
